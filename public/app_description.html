D U N K I N G C L U B

Detailed instructions for registering and using the program.

Welcome to our program!

Below you'll find some key features illustrated with images.

When you hover over the image, you'll get a hint.

You can also view a detailed test report by following the link.

Authentication

All required fields must be filled!

Enter Email:
The email must follow the format example@domain.com, without any special characters.

Select Continent and Country:
The selections is required to enable the next section.

Enter City Code:
The length depends on the country and may include numbers and letters.

Enter Password:
The password field must contain exactly four digits.

Register or Login:
Upon correct completion, the user will be redirected to the homepage, or the system will display errors and suggest corrections.
If it's the first registration, the user will be redirected to the registration page.

Choose an Action


Return to activation
The button is clicked by the user if they have entered an incorrect email address or password, or selected the wrong country and city code, in which case the user is returned to the authentication screen to correct the error..

Continue registration
The button is pressed by the user if he registers for the first time and wants to continue registration.
